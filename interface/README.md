# Awale Arena interface

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Preview dist

```sh
npm run preview
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

### Docker

```sh
docker build -t awale-arena-interface:latest .
```

```sh
docker run -d -p 8080:80 --name awale-arena-interface awale-arena-interface:latest
```
